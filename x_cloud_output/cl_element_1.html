    2. 2. Register the App

    In this step, we’ll register the app in the Cloud Admin so that we have the info needed to create the manifest.

    1. Open the Cloud Admin
    2. Click the APPS tab and click Create New App.

    3. Enter your account email in the App Name field, for example, myemail@email.com.

    4. Click Save.
    Tip: App names must be unique. Using your account email as a name for this tutorial ensures this name will be unique.

    ​We now have a Client ID and Secret for the app (located in the card on the left). We’ll need the Client ID when we create the element’s manifest.
